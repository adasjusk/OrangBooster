How to build your own booster copy
==========================================================================

1. Download and Install Python
------------------------------
a. Go to the official Python website: https://www.python.org/downloads/
b. Click "Download Python" (choose the latest version for Windows).
c. Run the downloaded installer.
d. IMPORTANT: On the first screen, check the box that says "Add Python to PATH".
e. Click "Install Now" and follow the prompts to complete the installation.

2. Extract OrangeBooster-main.zip
---------------------------------
a. Locate the `OrangeBooster-main.zip` file you downloaded (usually in your Downloads folder).
b. Right-click on `OrangeBooster-main.zip` and select "Extract All...".
c. In the dialog that appears, choose a destination folder (for example, your Desktop).
d. Click "Extract". This will create a folder named `OrangeBooster-main` in the chosen location.

3. Verify Python Installation
-----------------------------
a. Open Command Prompt (press Win+R, type `cmd`, and press Enter).
b. Type: python --version
You should see the installed Python version (e.g., Python 3.11.x).

4. Run builder.bat
------------------
a. In Command Prompt, navigate to the extracted `OrangeBooster-main` folder. For example:

    cd C:\Users\adasjusk\Desktop\OrangeBooster-main

b. Run the batch file by typing:

    builder.bat

Troubleshooting
---------------
- If you get a "python is not recognized" error, ensure Python is added to your PATH (see step 1d).
- If `builder.bat` fails, check for error messages and ensure all required files are present in the `OrangeBooster-main` folder.
- If problems caused with your computer it's not developers fault
- if any big issues or old code give a issue on: https://github.com/adasjusk/OrangBooster/issues

You're done!
